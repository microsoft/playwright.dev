`Suite` is a group of tests. All tests in Playwright Test form the following hierarchy:
* Root suite has a child suite for each [TestProject].
  * Project suite #1. Has a child suite for each test file in the project.
    * File suite #1
      * [TestCase] #1
      * [TestCase] #2
      * Suite corresponding to a [test.describe(title, callback)](./api/class-test.mdx#test-describe) group
        * [TestCase] #1 in a group
        * [TestCase] #2 in a group
      * < more test cases ... >
    * File suite #2
    * < more file suites ... >
  * Project suite #2
  * < more project suites ... >

Reporter is given a root suite in the [reporter.onBegin(config, suite)](./api/class-reporter.mdx#reporter-on-begin) method.

## suite.allTests() {#suite-all-tests}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="suite-all-tests-return"/> &#60;[Array]<[TestCase]>&#62;<a href="#suite-all-tests-return" class="list-anchor">#</a>

Returns the list of all test cases in this suite and its descendants, as opposite to [suite.tests](./api/class-suite.mdx#suite-tests).

## suite.titlePath() {#suite-title-path}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="suite-title-path-return"/> &#60;[Array]<[string]>&#62;<a href="#suite-title-path-return" class="list-anchor">#</a>

Returns a list of titles from the root down to this suite.

## suite.location {#suite-location}
- type: &#60;[void]|[Location]&#62;

Location in the source where the suite is defined. Missing for root and project suites.

## suite.suites {#suite-suites}
- type: &#60;[Array]<[Suite]>&#62;

Child suites. See [Suite] for the hierarchy of suites.

## suite.tests {#suite-tests}
- type: &#60;[Array]<[TestCase]>&#62;

Test cases in the suite. Note that only test cases defined directly in this suite are in the list. Any test cases defined in nested [test.describe(title, callback)](./api/class-test.mdx#test-describe) groups are listed in the child [suite.suites](./api/class-suite.mdx#suite-suites).

## suite.title {#suite-title}
- type: &#60;[string]&#62;

Suite title.
* Empty for root suite.
* Project name for project suite.
* File path for file suite.
* Title passed to [test.describe(title, callback)](./api/class-test.mdx#test-describe) for a group suite.
