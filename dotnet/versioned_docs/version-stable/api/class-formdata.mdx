## FormData.Set(name, value) {#form-data-set}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.18</font>

- `name`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="form-data-set-option-name"/> &#60;[string]&#62; Field name.<a href="#form-data-set-option-name" class="list-anchor">#</a>
- `value`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="form-data-set-option-value"/> &#60;[string]|[bool]|[int]|Value&#62; Field value.<a href="#form-data-set-option-value" class="list-anchor">#</a>
  - `Name` &#60;[string]&#62; File name
  - `MimeType` &#60;[string]&#62; File type
  - `Buffer` &#60;[byte]&#91;&#93;&#62; File content
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="form-data-set-return"/> &#60;[FormData]&#62;<a href="#form-data-set-return" class="list-anchor">#</a>

Sets a field on the form. File values can be passed either as `Path` or as `FilePayload`.
