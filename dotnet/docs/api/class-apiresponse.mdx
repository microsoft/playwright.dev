## ApiResponse.BodyAsync() {#api-response-body}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-body-return"/> &#60;[byte]&#91;&#93;&#62;<a href="#api-response-body-return" class="list-anchor">#</a>

Returns the buffer with response body.

## ApiResponse.DisposeAsync() {#api-response-dispose}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-dispose-return"/> &#60;[void]&#62;<a href="#api-response-dispose-return" class="list-anchor">#</a>

Disposes the body of this response. If not called then the body will stay in memory until the context closes.

## ApiResponse.Headers {#api-response-headers}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-headers-return"/> &#60;[IDictionary]<[string], [string]>&#62;<a href="#api-response-headers-return" class="list-anchor">#</a>

An object with all the response HTTP headers associated with this response.

## ApiResponse.HeadersArray {#api-response-headers-array}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-headers-array-return"/> &#60;[IEnumerable]<`HttpHeader`>&#62;<a href="#api-response-headers-array-return" class="list-anchor">#</a>
  - `name` &#60;[string]&#62; Name of the header.
  - `value` &#60;[string]&#62; Value of the header.

An array with all the request HTTP headers associated with this response. Header names are not lower-cased. Headers with multiple entries, such as `Set-Cookie`, appear in the array multiple times.

## ApiResponse.JsonAsync() {#api-response-json}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-json-return"/> &#60;[JsonElement]?&#62;<a href="#api-response-json-return" class="list-anchor">#</a>

Returns the JSON representation of response body.

This method will throw if the response body is not parsable via `JSON.parse`.

## ApiResponse.Ok {#api-response-ok}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-ok-return"/> &#60;[bool]&#62;<a href="#api-response-ok-return" class="list-anchor">#</a>

Contains a boolean stating whether the response was successful (status in the range 200-299) or not.

## ApiResponse.Status {#api-response-status}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-status-return"/> &#60;[int]&#62;<a href="#api-response-status-return" class="list-anchor">#</a>

Contains the status code of the response (e.g., 200 for a success).

## ApiResponse.StatusText {#api-response-status-text}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-status-text-return"/> &#60;[string]&#62;<a href="#api-response-status-text-return" class="list-anchor">#</a>

Contains the status text of the response (e.g. usually an "OK" for a success).

## ApiResponse.TextAsync() {#api-response-text}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-text-return"/> &#60;[string]&#62;<a href="#api-response-text-return" class="list-anchor">#</a>

Returns the text representation of response body.
