### ShowChapterAsync {#screencast-show-chapter}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>screencast.ShowChapterAsync</x-search>

Shows a chapter overlay with a title and optional description, centered on the page with a blurred backdrop. Useful for narrating video recordings. The overlay is removed after the specified duration, or 2000ms.

**Usage**

```csharp
await Screencast.ShowChapterAsync(title, options);
```

**Arguments**
- `title` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-chapter-option-title"/><a href="#screencast-show-chapter-option-title" class="list-anchor">#</a>
  
  Title text displayed prominently in the overlay.
- `options` `ScreencastShowChapterOptions?` *(optional)*
  - `Description` [string]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-chapter-option-description"/><a href="#screencast-show-chapter-option-description" class="list-anchor">#</a>
    
    Optional description text displayed below the title.
  - `Duration` [float]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-chapter-option-duration"/><a href="#screencast-show-chapter-option-duration" class="list-anchor">#</a>
    
    Duration in milliseconds after which the overlay is automatically removed. Defaults to `2000`.

**Returns**
- [void]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-chapter-return"/><a href="#screencast-show-chapter-return" class="list-anchor">#</a>

### ShowOverlayAsync {#screencast-show-overlay}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>screencast.ShowOverlayAsync</x-search>

Adds an overlay with the given HTML content. The overlay is displayed on top of the page until removed. Returns a disposable that removes the overlay when disposed.

**Usage**

```csharp
await Screencast.ShowOverlayAsync(html, options);
```

**Arguments**
- `html` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-overlay-option-html"/><a href="#screencast-show-overlay-option-html" class="list-anchor">#</a>
  
  HTML content for the overlay.
- `options` `ScreencastShowOverlayOptions?` *(optional)*
  - `Duration` [float]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-overlay-option-duration"/><a href="#screencast-show-overlay-option-duration" class="list-anchor">#</a>
    
    Duration in milliseconds after which the overlay is automatically removed. Overlay stays until dismissed if not provided.

**Returns**
- [Disposable]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-show-overlay-return"/><a href="#screencast-show-overlay-return" class="list-anchor">#</a>

### StartAsync {#screencast-start}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>screencast.StartAsync</x-search>

Starts the screencast. When [Path](/api/class-screencast.mdx#screencast-start-option-path) is provided, it saves video recording to the specified file. When [OnFrame](/api/class-screencast.mdx#screencast-start-option-on-frame) is provided, delivers JPEG-encoded frames to the callback. Both can be used together.

**Usage**

**Arguments**
- `options` `ScreencastStartOptions?` *(optional)*
  - `OnFrame` [Func]&lt;[ScreencastFrame], [Task]&gt; *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-start-option-on-frame"/><a href="#screencast-start-option-on-frame" class="list-anchor">#</a>
    - `Data` [byte]&#91;&#93;
      
      JPEG-encoded frame data.
    
    Callback that receives JPEG-encoded frame data.
  - `Path` [string]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-start-option-path"/><a href="#screencast-start-option-path" class="list-anchor">#</a>
    
    Path where the video should be saved when the screencast is stopped. When provided, video recording is started.
  - `Quality` [int]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-start-option-quality"/><a href="#screencast-start-option-quality" class="list-anchor">#</a>
    
    The quality of the image, between 0-100.

**Returns**
- [Disposable]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="screencast-start-return"/><a href="#screencast-start-return" class="list-anchor">#</a>

### StopAsync {#screencast-stop}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>screencast.StopAsync</x-search>

Stops the screencast and video recording if active. If a video was being recorded, saves it to the path specified in [Screencast.StartAsync()](/api/class-screencast.mdx#screencast-start).

**Usage**

```csharp
await Screencast.StopAsync();
```

**Returns**
