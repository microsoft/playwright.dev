The [WebSocket] class represents websocket connections in the page.


---

## Events

### event Close {#web-socket-event-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.8</font>


Fired when the websocket closes.

**Usage**

```csharp
WebSocket.Close += async data => {};
```

**Event data**
- &#60;[WebSocket]&#62;

---

### event FrameReceived {#web-socket-event-frame-received}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>


Fired when the websocket receives a frame.

**Usage**

```csharp
WebSocket.FrameReceived += async data => {};
```

**Event data**
- &#60;[WebSocketFrame]&#62;

---

### event FrameSent {#web-socket-event-frame-sent}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>


Fired when the websocket sends a frame.

**Usage**

```csharp
WebSocket.FrameSent += async data => {};
```

**Event data**
- &#60;[WebSocketFrame]&#62;

---

### event SocketError {#web-socket-event-socket-error}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>


Fired when the websocket has an error.

**Usage**

```csharp
WebSocket.SocketError += async data => {};
```

**Event data**
- &#60;[String]&#62;
