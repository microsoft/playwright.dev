[FileChooser] objects are dispatched by the page in the [Page.FileChooser](/api/class-page.mdx#page-event-file-chooser) event.

```csharp
var fileChooser = await page.RunAndWaitForFileChooserAsync(async () =>
{
    await page.GetByText("Upload").ClickAsync();
});
await fileChooser.SetFilesAsync("temp.txt");
```

## SetFilesAsync {#file-chooser-set-files}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.8</font>


Sets the value of the file input this chooser is associated with. If some of the `filePaths` are relative paths, then they are resolved relative to the current working directory. For empty array, clears the selected files.

**Usage**

```csharp
await FileChooser.SetFilesAsync(files, options);
```

**Parameters**
- `files` &#60;[string]|[IEnumerable]<[string]>|`FilePayload`|[IEnumerable]<`FilePayload`>&#62;<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="file-chooser-set-files-option-files"/><a href="#file-chooser-set-files-option-files" class="list-anchor">#</a>
  - `Name` &#60;[string]&#62; File name
  - `MimeType` &#60;[string]&#62; File type
  - `Buffer` &#60;[byte]&#91;&#93;&#62; File content
- `options` &#60;`FileChooserSetFilesOptions?`&#62;
  - `NoWaitAfter` &#60;[bool]?&#62; Actions that initiate navigations are waiting for these navigations to happen and for pages to start loading. You can opt out of waiting via setting this flag. You would only need this option in the exceptional cases such as navigating to inaccessible pages. Defaults to `false`.<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="file-chooser-set-files-option-no-wait-after"/><a href="#file-chooser-set-files-option-no-wait-after" class="list-anchor">#</a>
  - `Timeout` &#60;[double]?&#62; Maximum time in milliseconds, defaults to 30 seconds, pass `0` to disable timeout. The default value can be changed by using the [BrowserContext.SetDefaultTimeout();](/api/class-browsercontext.mdx#browser-context-set-default-timeout) or [Page.SetDefaultTimeout();](/api/class-page.mdx#page-set-default-timeout) methods.<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="file-chooser-set-files-option-timeout"/><a href="#file-chooser-set-files-option-timeout" class="list-anchor">#</a>
