The [WebSocketFrame] class represents frames sent over [WebSocket] connections in the page. Frame payload is returned by either [WebSocketFrame.Text](/api/class-websocketframe.mdx#web-socket-frame-text) or [WebSocketFrame.Binary](/api/class-websocketframe.mdx#web-socket-frame-binary) method depending on the its type.


- [WebSocketFrame.Binary](/api/class-websocketframe.mdx#web-socket-frame-binary)
- [WebSocketFrame.Text](/api/class-websocketframe.mdx#web-socket-frame-text)

## WebSocketFrame.Binary {#web-socket-frame-binary}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="web-socket-frame-binary-return"/> &#60;[byte]&#91;&#93;?&#62;<a href="#web-socket-frame-binary-return" class="list-anchor">#</a>

Returns binary payload.

## WebSocketFrame.Text {#web-socket-frame-text}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="web-socket-frame-text-return"/> &#60;[string]?&#62;<a href="#web-socket-frame-text-return" class="list-anchor">#</a>

Returns text payload.
