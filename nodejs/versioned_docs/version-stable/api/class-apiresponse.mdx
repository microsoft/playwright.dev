[APIResponse] class represents responses returned by [apiRequestContext.get()](/api/class-apirequestcontext.mdx#api-request-context-get) and similar methods.


---

## Methods

### body {#api-response-body}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.body</x-search>

Returns the buffer with response body.

**Usage**

```js
await apiResponse.body();
```

**Returns**
- [Promise]<[Buffer]><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-body-return"/><a href="#api-response-body-return" class="list-anchor">#</a>

---

### dispose {#api-response-dispose}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.dispose</x-search>

Disposes the body of this response. If not called then the body will stay in memory until the context closes.

**Usage**

```js
await apiResponse.dispose();
```

---

### headers {#api-response-headers}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.headers</x-search>

An object with all the response HTTP headers associated with this response.

**Usage**

```js
apiResponse.headers();
```

**Returns**
- [Object]<[string], [string]><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-headers-return"/><a href="#api-response-headers-return" class="list-anchor">#</a>

---

### headersArray {#api-response-headers-array}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.headersArray</x-search>

An array with all the request HTTP headers associated with this response. Header names are not lower-cased. Headers with multiple entries, such as `Set-Cookie`, appear in the array multiple times.

**Usage**

```js
apiResponse.headersArray();
```

**Returns**
- [Array]<[Object]><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-headers-array-return"/><a href="#api-response-headers-array-return" class="list-anchor">#</a>
  - `name` [string]
    
    Name of the header.
  - `value` [string]
    
    Value of the header.

This method will throw if the response body is not parsable via `JSON.parse`.

**Usage**

```js
await apiResponse.json();
```
