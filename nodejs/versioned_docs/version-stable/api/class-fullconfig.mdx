Resolved configuration which is accessible via [testInfo.config](/api/class-testinfo.mdx#test-info-config) and is passed to the test reporters. To see the format of Playwright configuration file, please see [TestConfig] instead.


---

## Properties

### configFile {#full-config-config-file}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.20</font><x-search>fullConfig.configFile</x-search>

Path to the configuration file used to run the tests. The value is an empty string if no config file was used.

**Usage**

```js
fullConfig.configFile
```

**Type**
- [string]

### reportSlowTests {#full-config-report-slow-tests}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>fullConfig.reportSlowTests</x-search>

See [testConfig.reportSlowTests](/api/class-testconfig.mdx#test-config-report-slow-tests).

**Usage**

```js
fullConfig.reportSlowTests
```

**Type**
- [null] | [Object]
  - `max` [number]
    
    The maximum number of slow test files to report. Defaults to `5`.
  - `threshold` [number]
    
    Test duration in milliseconds that is considered slow. Defaults to 15 seconds.
