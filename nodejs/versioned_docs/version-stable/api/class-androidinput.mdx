## Methods

### drag {#android-input-drag}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidInput.drag</x-search>

Performs a drag between `from` and `to` points.

**Usage**

```js
await androidInput.drag(from, to, steps);
```

**Arguments**
- `from` [Object]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-input-drag-option-from"/><a href="#android-input-drag-option-from" class="list-anchor">#</a>
  - `x` [number]
    
    
  - `y` [number]
    
    
  The start point of the drag.
- `to` [Object]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-input-drag-option-to"/><a href="#android-input-drag-option-to" class="list-anchor">#</a>
  - `x` [number]
    
    
  - `y` [number]
    
    
  The end point of the drag.
- `steps` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-input-drag-option-steps"/><a href="#android-input-drag-option-steps" class="list-anchor">#</a>
  
  The number of steps in the drag. Each step takes 5 milliseconds to complete.

### swipe {#android-input-swipe}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidInput.swipe</x-search>

Swipes following the path defined by `segments`.

**Usage**

```js
await androidInput.swipe(from, segments, steps);
```

**Arguments**
- `from` [Object]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-input-swipe-option-from"/><a href="#android-input-swipe-option-from" class="list-anchor">#</a>
  - `x` [number]
    
    
  - `y` [number]
    
    
  The point to start swiping from.
- `segments` [Array]<[Object]><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-input-swipe-option-segments"/><a href="#android-input-swipe-option-segments" class="list-anchor">#</a>
  - `x` [number]
    
    
  - `y` [number]
    
    
  Points following the `from` point in the swipe gesture.
- `steps` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-input-swipe-option-steps"/><a href="#android-input-swipe-option-steps" class="list-anchor">#</a>
  
  The number of steps for each segment. Each step takes 5 milliseconds to complete, so 100 steps means half a second per each segment.
