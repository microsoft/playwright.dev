[AndroidSocket] is a way to communicate with a process launched on the [AndroidDevice]. Use [androidDevice.open()](/api/class-androiddevice.mdx#android-device-open) to open a socket.


---

## Methods

### close {#android-socket-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidSocket.close</x-search>

Closes the socket.

**Usage**

```js
await androidSocket.close();
```

**Returns**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-socket-close-return"/><a href="#android-socket-close-return" class="list-anchor">#</a>

---

### write {#android-socket-write}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidSocket.write</x-search>

Writes some [data](/api/class-androidsocket.mdx#android-socket-write-option-data) to the socket.

**Usage**

```js
await androidSocket.write(data);
```

**Arguments**
- `data` [Buffer]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-socket-write-option-data"/><a href="#android-socket-write-option-data" class="list-anchor">#</a>
  
  Data to write.

**Returns**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-socket-write-return"/><a href="#android-socket-write-return" class="list-anchor">#</a>

## Events

### on('close') {#android-socket-event-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidSocket.on('close')</x-search>

Emitted when the socket is closed.

**Usage**

```js
androidSocket.on('close', data => {});
```

---

### on('data') {#android-socket-event-data}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidSocket.on('data')</x-search>

Emitted when data is available to read from the socket.

**Usage**

```js
androidSocket.on('data', data => {});
```

- [Buffer]
