`WorkerInfo` contains information about the worker that is running tests and is available to worker-scoped fixtures. `WorkerInfo` is a subset of [TestInfo] that is available in many other places.


---

## Properties

### config {#worker-info-config}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>workerInfo.config</x-search>

Processed configuration from the [configuration file](../test-configuration.mdx).

**Usage**

```js
workerInfo.config
```

**Type**
- [TestConfig]

---

### parallelIndex {#worker-info-parallel-index}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>workerInfo.parallelIndex</x-search>

The index of the worker between `0` and `workers - 1`. It is guaranteed that workers running at the same time have a different `parallelIndex`. When a worker is restarted, for example after a failure, the new worker process has the same `parallelIndex`.

Also available as `process.env.TEST_PARALLEL_INDEX`. Learn more about [parallelism and sharding](../test-parallel.mdx) with Playwright Test.

**Usage**

```js
workerInfo.parallelIndex
```

**Type**
- [number]

---

### project {#worker-info-project}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>workerInfo.project</x-search>

Processed project configuration from the [configuration file](../test-configuration.mdx).

**Usage**

```js
workerInfo.project
```

**Type**
- [TestProject]

---

### workerIndex {#worker-info-worker-index}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>workerInfo.workerIndex</x-search>

The unique index of the worker process that is running the test. When a worker is restarted, for example after a failure, the new worker process gets a new unique `workerIndex`.

Also available as `process.env.TEST_WORKER_INDEX`. Learn more about [parallelism and sharding](../test-parallel.mdx) with Playwright Test.

**Usage**

```js
workerInfo.workerIndex
```

**Type**
- [number]
