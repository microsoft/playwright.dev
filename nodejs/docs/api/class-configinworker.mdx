### reportSlowTests {#config-in-worker-report-slow-tests}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>configInWorker.reportSlowTests</x-search>

See [testConfig.reportSlowTests](/api/class-testconfig.mdx#test-config-report-slow-tests).

**Usage**

```js
configInWorker.reportSlowTests
```

**Type**
- [null]|[Object]
  - `max` [number]
    
    The maximum number of slow test files to report. Defaults to `5`.
  - `threshold` [number]
    
    Test duration in milliseconds that is considered slow. Defaults to 15 seconds.
