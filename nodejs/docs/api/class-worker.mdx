The Worker class represents a [WebWorker](https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API). `worker` event is emitted on the page object to signal a worker creation. `close` event is emitted on the worker object when the worker is gone.

```js
page.on('worker', worker => {
  console.log('Worker created: ' + worker.url());
  worker.on('close', worker => console.log('Worker destroyed: ' + worker.url()));
});

console.log('Current workers:');
for (const worker of page.workers())
  console.log('  ' + worker.url());
```

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.8</font>

- type: &#60;[Worker]&#62;

Emitted when this dedicated [WebWorker](https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API) is terminated.

## worker.evaluate(pageFunction[, arg]) {#worker-evaluate}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.8</font>

- `pageFunction`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="worker-evaluate-option-expression"/> &#60;[function]|[string]&#62; Function to be evaluated in the worker context.<a href="#worker-evaluate-option-expression" class="list-anchor">#</a>
- `arg?`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="worker-evaluate-option-arg"/> &#60;[EvaluationArgument]&#62; Optional argument to pass to `pageFunction`.<a href="#worker-evaluate-option-arg" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="worker-evaluate-return"/> &#60;[Promise]<[Serializable]>&#62;<a href="#worker-evaluate-return" class="list-anchor">#</a>

Returns the return value of `pageFunction`.

If the function passed to the [worker.evaluate(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate) returns a [Promise], then [worker.evaluate(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate) would wait for the promise to resolve and return its value.

If the function passed to the [worker.evaluate(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate) returns a non-[Serializable] value, then [worker.evaluate(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate) returns `undefined`. Playwright also supports transferring some additional values that are not serializable by `JSON`: `-0`, `NaN`, `Infinity`, `-Infinity`.

## worker.evaluateHandle(pageFunction[, arg]) {#worker-evaluate-handle}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.8</font>

- `pageFunction`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="worker-evaluate-handle-option-expression"/> &#60;[function]|[string]&#62; Function to be evaluated in the worker context.<a href="#worker-evaluate-handle-option-expression" class="list-anchor">#</a>
- `arg?`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="worker-evaluate-handle-option-arg"/> &#60;[EvaluationArgument]&#62; Optional argument to pass to `pageFunction`.<a href="#worker-evaluate-handle-option-arg" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="worker-evaluate-handle-return"/> &#60;[Promise]<[JSHandle]>&#62;<a href="#worker-evaluate-handle-return" class="list-anchor">#</a>

Returns the return value of `pageFunction` as a [JSHandle].

The only difference between [worker.evaluate(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate) and [worker.evaluateHandle(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate-handle) is that [worker.evaluateHandle(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate-handle) returns [JSHandle].

If the function passed to the [worker.evaluateHandle(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate-handle) returns a [Promise], then [worker.evaluateHandle(pageFunction[, arg])](/api/class-worker.mdx#worker-evaluate-handle) would wait for the promise to resolve and return its value.
