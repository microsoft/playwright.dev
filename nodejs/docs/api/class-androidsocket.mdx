[AndroidSocket] is a way to communicate with a process launched on the [AndroidDevice]. Use [androidDevice.open(command)](/api/class-androiddevice.mdx#android-device-open) to open a socket.


- [androidSocket.on('close')](/api/class-androidsocket.mdx#android-socket-event-close)
- [androidSocket.on('data')](/api/class-androidsocket.mdx#android-socket-event-data)
- [androidSocket.close()](/api/class-androidsocket.mdx#android-socket-close)
- [androidSocket.write(data)](/api/class-androidsocket.mdx#android-socket-write)

## androidSocket.on('close') {#android-socket-event-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

Emitted when the socket is closed.

## androidSocket.on('data') {#android-socket-event-data}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- type: &#60;[Buffer]&#62;

Emitted when data is available to read from the socket.

## androidSocket.close() {#android-socket-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-socket-close-return"/> &#60;[Promise]<[void]>&#62;<a href="#android-socket-close-return" class="list-anchor">#</a>

Closes the socket.

## androidSocket.write(data) {#android-socket-write}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- `data`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-socket-write-option-data"/> &#60;[Buffer]&#62; Data to write.<a href="#android-socket-write-option-data" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-socket-write-return"/> &#60;[Promise]<[void]>&#62;<a href="#android-socket-write-return" class="list-anchor">#</a>

Writes some `data` to the socket.
