Coverage gathers information about parts of JavaScript and CSS that were used by the page.

An example of using JavaScript coverage to produce Istanbul report for page load:

:::note
Coverage APIs are only supported on Chromium-based browsers.
:::

```js
const { chromium } = require('playwright');
const v8toIstanbul = require('v8-to-istanbul');

(async() => {
  const browser = await chromium.launch();
  const page = await browser.newPage();
  await page.coverage.startJSCoverage();
  await page.goto('https://chromium.org');
  const coverage = await page.coverage.stopJSCoverage();
  for (const entry of coverage) {
    const converter = v8toIstanbul('', 0, { source: entry.source });
    await converter.load();
    converter.applyCoverage(entry.functions);
    console.log(JSON.stringify(converter.toIstanbul()));
  }
  await browser.close();
})();
```

  - `resetOnNavigation?`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="coverage-start-css-coverage-option-reset-on-navigation"/> &#60;[boolean]&#62; Whether to reset coverage on every navigation. Returns coverage is started

## coverage.startJSCoverage([options]) {#coverage-start-js-coverage}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font>

- `options?` &#60;[Object]&#62;
  - `reportAnonymousScripts?`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="coverage-start-js-coverage-option-report-anonymous-scripts"/> &#60;[boolean]&#62; Whether anonymous scripts generated by the page should be reported. Defaults to `false`.<a href="#coverage-start-js-coverage-option-report-anonymous-scripts" class="list-anchor">#</a>
  - `resetOnNavigation?`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="coverage-start-js-coverage-option-reset-on-navigation"/> &#60;[boolean]&#62; Whether to reset coverage on every navigation. Defaults to `true`.<a href="#coverage-start-js-coverage-option-reset-on-navigation" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="coverage-start-js-coverage-return"/> &#60;[Promise]<[void]>&#62;<a href="#coverage-start-js-coverage-return" class="list-anchor">#</a>

Returns coverage is started

:::note
Anonymous scripts are ones that don't have an associated url. These are scripts that are dynamically created on the page using `eval` or `new Function`. If `reportAnonymousScripts` is set to `true`, anonymous scripts will have `__playwright_evaluation_script__` as their URL.
:::

## coverage.stopCSSCoverage() {#coverage-stop-css-coverage}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="coverage-stop-css-coverage-return"/> &#60;[Promise]<[Array]<[Object]>>&#62;<a href="#coverage-stop-css-coverage-return" class="list-anchor">#</a>
  - `url` &#60;[string]&#62; StyleSheet URL
  - `text?` &#60;[string]&#62; StyleSheet content, if available.
  - `ranges` &#60;[Array]<[Object]>&#62; StyleSheet ranges that were used. Ranges are sorted and non-overlapping.
    - `start` &#60;[number]&#62; A start offset in text, inclusive
    - `end` &#60;[number]&#62; An end offset in text, exclusive

Returns the array of coverage reports for all stylesheets

:::note
CSS Coverage doesn't include dynamically injected style tags without sourceURLs.
:::

## coverage.stopJSCoverage() {#coverage-stop-js-coverage}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="coverage-stop-js-coverage-return"/> &#60;[Promise]<[Array]<[Object]>>&#62;<a href="#coverage-stop-js-coverage-return" class="list-anchor">#</a>
  - `url` &#60;[string]&#62; Script URL
  - `scriptId` &#60;[string]&#62; Script ID
  - `source?` &#60;[string]&#62; Script content, if applicable.
  - `functions` &#60;[Array]<[Object]>&#62; V8-specific coverage format.
    - `functionName` &#60;[string]&#62;
    - `isBlockCoverage` &#60;[boolean]&#62;
    - `ranges` &#60;[Array]<[Object]>&#62;
      - `count` &#60;[number]&#62;
      - `startOffset` &#60;[number]&#62;
      - `endOffset` &#60;[number]&#62;

Returns the array of coverage reports for all scripts

:::note
JavaScript Coverage doesn't include anonymous scripts by default. However, scripts with sourceURLs are reported.
:::
