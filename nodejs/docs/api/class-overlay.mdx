Interface for managing page overlays that display persistent visual indicators on top of the page.


---

## Methods

### chapter {#overlay-chapter}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>overlay.chapter</x-search>

Shows a chapter overlay with a title and optional description, centered on the page with a blurred backdrop. Useful for narrating video recordings. The overlay is removed after the specified duration, or 2000ms.

**Usage**

```js
await overlay.chapter(title);
await overlay.chapter(title, options);
```

**Arguments**
- `title` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-option-title"/><a href="#overlay-chapter-option-title" class="list-anchor">#</a>
  
  Title text displayed prominently in the overlay.
- `options` [Object] *(optional)*
  - `description` [string] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-option-description"/><a href="#overlay-chapter-option-description" class="list-anchor">#</a>
    
    Optional description text displayed below the title.
  - `duration` [number] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-option-duration"/><a href="#overlay-chapter-option-duration" class="list-anchor">#</a>
    
    Duration in milliseconds after which the overlay is automatically removed. Defaults to `2000`.

**Returns**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-return"/><a href="#overlay-chapter-return" class="list-anchor">#</a>

### setVisible {#overlay-set-visible}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>overlay.setVisible</x-search>

Sets visibility of all overlays without removing them.

**Usage**

```js
await overlay.setVisible(visible);
```

**Arguments**
- `visible` [boolean]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-set-visible-option-visible"/><a href="#overlay-set-visible-option-visible" class="list-anchor">#</a>
  
  Whether overlays should be visible.

**Returns**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-set-visible-return"/><a href="#overlay-set-visible-return" class="list-anchor">#</a>

### show {#overlay-show}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>overlay.show</x-search>

Adds an overlay with the given HTML content. The overlay is displayed on top of the page until removed. Returns a disposable that removes the overlay when disposed.

**Usage**

```js
await overlay.show(html);
await overlay.show(html, options);
```

**Arguments**
- `html` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-show-option-html"/><a href="#overlay-show-option-html" class="list-anchor">#</a>
  
  HTML content for the overlay.
- `options` [Object] *(optional)*
  - `duration` [number] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-show-option-duration"/><a href="#overlay-show-option-duration" class="list-anchor">#</a>
    
    Duration in milliseconds after which the overlay is automatically removed. Overlay stays until dismissed if not provided.

**Returns**
- [Promise]&lt;[Disposable]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-show-return"/><a href="#overlay-show-return" class="list-anchor">#</a>
