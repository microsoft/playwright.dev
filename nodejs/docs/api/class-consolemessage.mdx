[ConsoleMessage] objects are dispatched by page via the [page.on('console')](./api/class-page.mdx#page-event-console) event.


- [consoleMessage.args()](./api/class-consolemessage.mdx#console-message-args)
- [consoleMessage.location()](./api/class-consolemessage.mdx#console-message-location)
- [consoleMessage.text()](./api/class-consolemessage.mdx#console-message-text)
- [consoleMessage.type()](./api/class-consolemessage.mdx#console-message-type)

## consoleMessage.args() {#console-message-args}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="console-message-args-return"/> &#60;[Array]<[JSHandle]>&#62;<a href="#console-message-args-return" class="list-anchor">#</a>

List of arguments passed to a `console` function call. See also [page.on('console')](./api/class-page.mdx#page-event-console).

## consoleMessage.location() {#console-message-location}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="console-message-location-return"/> &#60;[Object]&#62;<a href="#console-message-location-return" class="list-anchor">#</a>
  - `url` &#60;[string]&#62; URL of the resource.
  - `lineNumber` &#60;[number]&#62; 0-based line number in the resource.
  - `columnNumber` &#60;[number]&#62; 0-based column number in the resource.

## consoleMessage.text() {#console-message-text}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="console-message-text-return"/> &#60;[string]&#62;<a href="#console-message-text-return" class="list-anchor">#</a>

The text of the console message.

## consoleMessage.type() {#console-message-type}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="console-message-type-return"/> &#60;[string]&#62;<a href="#console-message-type-return" class="list-anchor">#</a>

One of the following values: `'log'`, `'debug'`, `'info'`, `'error'`, `'warning'`, `'dir'`, `'dirxml'`, `'table'`, `'trace'`, `'clear'`, `'startGroup'`, `'startGroupCollapsed'`, `'endGroup'`, `'assert'`, `'profile'`, `'profileEnd'`, `'count'`, `'timeEnd'`.
