[AndroidWebView] represents a WebView open on the [AndroidDevice]. WebView is usually obtained using [androidDevice.webView()](/api/class-androiddevice.mdx#android-device-web-view).


---

## Methods

### page {#android-web-view-page}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>androidWebView.page</x-search>

Connects to the WebView and returns a regular Playwright [Page] to interact with.

**Usage**

```js
await androidWebView.page();
```

**Returns**
- [Promise]&lt;[Page]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-web-view-page-return"/><a href="#android-web-view-page-return" class="list-anchor">#</a>
