[AndroidWebView] represents a WebView open on the [AndroidDevice]. WebView is usually obtained using [androidDevice.webView(selector[, options])](/api/class-androiddevice.mdx#android-device-web-view).


- [androidWebView.on('close')](/api/class-androidwebview.mdx#android-web-view-event-close)
- [androidWebView.page()](/api/class-androidwebview.mdx#android-web-view-page)
- [androidWebView.pid()](/api/class-androidwebview.mdx#android-web-view-pid)
- [androidWebView.pkg()](/api/class-androidwebview.mdx#android-web-view-pkg)

## androidWebView.on('close') {#android-web-view-event-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

Emitted when the WebView is closed.

## androidWebView.page() {#android-web-view-page}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-web-view-page-return"/> &#60;[Promise]<[Page]>&#62;<a href="#android-web-view-page-return" class="list-anchor">#</a>

Connects to the WebView and returns a regular Playwright [Page] to interact with.

## androidWebView.pid() {#android-web-view-pid}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

WebView process PID.

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-web-view-pkg-return"/> &#60;[string]&#62;<a href="#android-web-view-pkg-return" class="list-anchor">#</a>

WebView package identifier.
