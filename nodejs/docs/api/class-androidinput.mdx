## androidInput.drag(from, to, steps) {#android-input-drag}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- `from`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-drag-option-from"/> &#60;[Object]&#62; The start point of the drag.<a href="#android-input-drag-option-from" class="list-anchor">#</a>
  - `x` &#60;[number]&#62;
  - `y` &#60;[number]&#62;
- `to`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-drag-option-to"/> &#60;[Object]&#62; The end point of the drag.<a href="#android-input-drag-option-to" class="list-anchor">#</a>
  - `x` &#60;[number]&#62;
  - `y` &#60;[number]&#62;
- `steps`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-drag-option-steps"/> &#60;[number]&#62; The number of steps in the drag. Each step takes 5 milliseconds to complete.<a href="#android-input-drag-option-steps" class="list-anchor">#</a>

## androidInput.swipe(from, segments, steps) {#android-input-swipe}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font>

- `from`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-swipe-option-from"/> &#60;[Object]&#62; The point to start swiping from.<a href="#android-input-swipe-option-from" class="list-anchor">#</a>
  - `x` &#60;[number]&#62;
  - `y` &#60;[number]&#62;
- `segments`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-swipe-option-segments"/> &#60;[Array]<[Object]>&#62; Points following the `from` point in the swipe gesture.<a href="#android-input-swipe-option-segments" class="list-anchor">#</a>
  - `x` &#60;[number]&#62;
  - `y` &#60;[number]&#62;
- `steps`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-swipe-option-steps"/> &#60;[number]&#62; The number of steps for each segment. Each step takes 5 milliseconds to complete, so 100 steps means half a second per each segment.<a href="#android-input-swipe-option-steps" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="android-input-swipe-return"/> &#60;[Promise]<[void]>&#62;<a href="#android-input-swipe-return" class="list-anchor">#</a>

Swipes following the path defined by `segments`.
