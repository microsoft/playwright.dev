API for controlling the Playwright debugger. The debugger allows pausing script execution and inspecting the page. Obtain the debugger instance via [browser_context.debugger](/api/class-browsercontext.mdx#browser-context-debugger).


---

## Methods

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.next</x-search>

**Usage**

```python
debugger.next()
```

### request_pause {#debugger-request-pause}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.request_pause</x-search>

Configures the debugger to pause before the next action is executed.

Throws if the debugger is already paused. Use [debugger.next()](/api/class-debugger.mdx#debugger-next) or [debugger.run_to()](/api/class-debugger.mdx#debugger-run-to) to step while paused.

Note that [page.pause()](/api/class-page.mdx#page-pause) is equivalent to a "debugger" statement — it pauses execution at the call site immediately. On the contrary, [debugger.request_pause()](/api/class-debugger.mdx#debugger-request-pause) is equivalent to "pause on next statement" — it configures the debugger to pause before the next action is executed.

**Usage**

```python
debugger.request_pause()
```

**Returns**
- [NoneType]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-request-pause-return"/><a href="#debugger-request-pause-return" class="list-anchor">#</a>

### run_to {#debugger-run-to}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.run_to</x-search>

Resumes script execution and pauses when an action originates from the given source location. Throws if the debugger is not paused.

**Usage**

```python
debugger.run_to(location)
```

**Arguments**
- `location` [Dict]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-run-to-option-location"/><a href="#debugger-run-to-option-location" class="list-anchor">#</a>
  - `file` [str]
    
    
  - `line` [int] *(optional)*
    
    
  - `column` [int] *(optional)*
    
    
  The source location to pause at.

**Returns**
- [NoneType]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-run-to-return"/><a href="#debugger-run-to-return" class="list-anchor">#</a>

### paused_details {#debugger-paused-details}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.paused_details</x-search>

Returns details about the currently paused call. Returns `null` if the debugger is not paused.

**Usage**

```python
debugger.paused_details
```

**Returns**
- [NoneType] | [Dict]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-paused-details-return"/><a href="#debugger-paused-details-return" class="list-anchor">#</a>
