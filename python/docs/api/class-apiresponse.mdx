## api_response.body() {#api-response-body}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-body-return"/> &#60;[bytes]&#62;<a href="#api-response-body-return" class="list-anchor">#</a>

Returns the buffer with response body.

## api_response.dispose() {#api-response-dispose}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-dispose-return"/> &#60;[NoneType]&#62;<a href="#api-response-dispose-return" class="list-anchor">#</a>

Disposes the body of this response. If not called then the body will stay in memory until the context closes.

## api_response.headers {#api-response-headers}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-headers-return"/> &#60;[Dict]\[[str], [str]\]&#62;<a href="#api-response-headers-return" class="list-anchor">#</a>

An object with all the response HTTP headers associated with this response.

## api_response.headers_array {#api-response-headers-array}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-headers-array-return"/> &#60;[List]\[[Dict]\]&#62;<a href="#api-response-headers-array-return" class="list-anchor">#</a>
  - `name` &#60;[str]&#62; Name of the header.
  - `value` &#60;[str]&#62; Value of the header.

An array with all the request HTTP headers associated with this response. Header names are not lower-cased. Headers with multiple entries, such as `Set-Cookie`, appear in the array multiple times.

## api_response.json() {#api-response-json}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-json-return"/> &#60;[Serializable]&#62;<a href="#api-response-json-return" class="list-anchor">#</a>

Returns the JSON representation of response body.

This method will throw if the response body is not parsable via `JSON.parse`.

## api_response.ok {#api-response-ok}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-ok-return"/> &#60;[bool]&#62;<a href="#api-response-ok-return" class="list-anchor">#</a>

Contains a boolean stating whether the response was successful (status in the range 200-299) or not.

## api_response.status {#api-response-status}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-status-return"/> &#60;[int]&#62;<a href="#api-response-status-return" class="list-anchor">#</a>

Contains the status code of the response (e.g., 200 for a success).

## api_response.status_text {#api-response-status-text}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-status-text-return"/> &#60;[str]&#62;<a href="#api-response-status-text-return" class="list-anchor">#</a>

Contains the status text of the response (e.g. usually an "OK" for a success).

## api_response.text() {#api-response-text}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-text-return"/> &#60;[str]&#62;<a href="#api-response-text-return" class="list-anchor">#</a>

Returns the text representation of response body.

## api_response.url {#api-response-url}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="api-response-url-return"/> &#60;[str]&#62;<a href="#api-response-url-return" class="list-anchor">#</a>

Contains the URL of the response.
