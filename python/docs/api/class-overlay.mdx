Interface for managing page overlays that display persistent visual indicators on top of the page.


---

## Methods

### chapter {#overlay-chapter}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>overlay.chapter</x-search>

Shows a chapter overlay with a title and optional description, centered on the page with a blurred backdrop. Useful for narrating video recordings. The overlay is removed after the specified duration, or 2000ms.

**Usage**

```python
overlay.chapter(title)
overlay.chapter(title, **kwargs)
```

**Arguments**
- `title` [str]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-option-title"/><a href="#overlay-chapter-option-title" class="list-anchor">#</a>
  
  Title text displayed prominently in the overlay.
- `description` [str] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-option-description"/><a href="#overlay-chapter-option-description" class="list-anchor">#</a>
  
  Optional description text displayed below the title.
- `duration` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-option-duration"/><a href="#overlay-chapter-option-duration" class="list-anchor">#</a>
  
  Duration in milliseconds after which the overlay is automatically removed. Defaults to `2000`.

**Returns**
- [NoneType]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-chapter-return"/><a href="#overlay-chapter-return" class="list-anchor">#</a>

### set_visible {#overlay-set-visible}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>overlay.set_visible</x-search>

Sets visibility of all overlays without removing them.

**Usage**

```python
overlay.set_visible(visible)
```

**Arguments**
- `visible` [bool]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-set-visible-option-visible"/><a href="#overlay-set-visible-option-visible" class="list-anchor">#</a>
  
  Whether overlays should be visible.

**Returns**
- [NoneType]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-set-visible-return"/><a href="#overlay-set-visible-return" class="list-anchor">#</a>

### show {#overlay-show}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>overlay.show</x-search>

Adds an overlay with the given HTML content. The overlay is displayed on top of the page until removed. Returns a disposable that removes the overlay when disposed.

**Usage**

```python
overlay.show(html)
overlay.show(html, **kwargs)
```

**Arguments**
- `html` [str]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-show-option-html"/><a href="#overlay-show-option-html" class="list-anchor">#</a>
  
  HTML content for the overlay.
- `duration` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-show-option-duration"/><a href="#overlay-show-option-duration" class="list-anchor">#</a>
  
  Duration in milliseconds after which the overlay is automatically removed. Overlay stays until dismissed if not provided.

**Returns**
- [Disposable]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="overlay-show-return"/><a href="#overlay-show-return" class="list-anchor">#</a>
