* extends: [Exception]

Error is raised whenever certain operations are terminated abnormally, e.g. browser closes while [page.evaluate(expression, **kwargs)](./api/class-page.mdx#pageevaluateexpression-kwargs) is running. All Playwright exceptions inherit from this class.


- [error.message](./api/class-error.mdx#errormessage)
- [error.name](./api/class-error.mdx#errorname)
- [error.stack](./api/class-error.mdx#errorstack)

## error.message
- type: <[str]>

Message of the error.

## error.name
- type: <[str]>

Name of the error which got thrown inside the browser. Optional.

## error.stack
- type: <[str]>

Stack of the error which got thrown inside the browser. Optional.
