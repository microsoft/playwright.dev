## Methods

### delete {#video-delete}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font><x-search>video.delete</x-search>

Deletes the video file. Will wait for the video to finish if necessary.

**Usage**

```python
video.delete()
```

**Returns**
- [NoneType]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="video-delete-return"/><a href="#video-delete-return" class="list-anchor">#</a>

### path {#video-path}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>video.path</x-search>

Returns the file system path this video will be recorded to. The video is guaranteed to be written to the filesystem upon closing the browser context. This method throws when connected remotely.

**Usage**

```python
video.path()
```

**Returns**
- [pathlib.Path]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="video-path-return"/><a href="#video-path-return" class="list-anchor">#</a>

---

### save_as {#video-save-as}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font><x-search>video.save_as</x-search>

Saves the video to a user-specified path. It is safe to call this method while the video is still in progress, or after the page has closed. This method waits until the page is closed and the video is fully saved.

**Usage**

```python
video.save_as(path)
```

**Arguments**
- `path` [Union]\[[str], [pathlib.Path]\]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="video-save-as-option-path"/><a href="#video-save-as-option-path" class="list-anchor">#</a>
  
  Path where the video should be saved.

**Returns**
- [NoneType]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="video-save-as-return"/><a href="#video-save-as-return" class="list-anchor">#</a>
