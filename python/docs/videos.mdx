import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';


Playwright can record videos for all pages in a [browser context](./core-concepts.mdx#browser-contexts). Videos are saved upon context closure, so make sure to await [browser_context.close()](./api/class-browsercontext.mdx#browser_contextclose).

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
context = browser.new_context(record_video_dir="videos/")
# Make sure to close, so that videos are saved.
context.close()
```

</TabItem>
<TabItem value="async">

```py
context = await browser.new_context(record_video_dir="videos/")
# Make sure to await close, so that videos are saved.
await context.close()
```

</TabItem>
</Tabs>

You can also specify video size, it defaults to viewport size scaled down to fit 800x800.

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
context = browser.new_context(
    record_video_dir="videos/",
    record_video_size={"width": 1024, "height": 768}
)
```

</TabItem>
<TabItem value="async">

```py
context = await browser.new_context(
    record_video_dir="videos/",
    record_video_size={"width": 1024, "height": 768}
)
```

</TabItem>
</Tabs>

Saved video files will appear in the specified folder. They all have generated unique names. For the multi-page scenarios, you can access the video file associated with the page via the [page.video](./api/class-page.mdx#pagevideo).

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
path = page.video.path()
```

</TabItem>
<TabItem value="async">

```py
path = await page.video.path()
```

</TabItem>
</Tabs>

:::note
Note that the video is only available after the page or browser context is closed.
:::
