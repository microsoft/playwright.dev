## browserServer.on('close')

Emitted when the browser server closes.

## browserServer.close()
- returns: <[Promise]>

Closes the browser gracefully and makes sure the process is terminated.

## browserServer.kill()
- returns: <[Promise]>

Kills the browser process and waits for the process to exit.

## browserServer.process()
- returns: <[ChildProcess]>

Spawned browser application process.

## browserServer.wsEndpoint()
- returns: <[string]>

Browser websocket url.

Browser websocket endpoint which can be used as an argument to [browserType.connect(params)](./class-browsertype.md#browsertypeconnectparams) to establish connection to the browser.
