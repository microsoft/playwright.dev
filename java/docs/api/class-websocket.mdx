## WebSocket.onClose(handler) {#web-socket-event-close}
- type: &#60;[WebSocket]&#62;

Fired when the websocket closes.

## WebSocket.onFrameReceived(handler) {#web-socket-event-frame-received}
- type: &#60;[WebSocketFrame]&#62;

Fired when the websocket receives a frame.

## WebSocket.onFrameSent(handler) {#web-socket-event-frame-sent}
- type: &#60;[WebSocketFrame]&#62;

Fired when the websocket sends a frame.

## WebSocket.onSocketError(handler) {#web-socket-event-socket-error}
- type: &#60;[String]&#62;

Fired when the websocket has an error.

## WebSocket.waitForFrameReceived([options], callback) {#web-socket-wait-for-frame-received}
- `options` &#60;`WebSocket.WaitForFrameReceivedOptions`&#62;
  - `setPredicate`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-option-predicate"/> &#60;[Predicate]<[WebSocketFrame]>&#62; Receives the [WebSocketFrame] object and resolves to truthy value when the waiting should resolve.<a href="#web-socket-wait-for-frame-received-option-predicate" class="list-anchor">#</a>
  - `setTimeout`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-option-timeout"/> &#60;[double]&#62; Maximum time to wait for in milliseconds. Defaults to `30000` (30 seconds). Pass `0` to disable timeout. The default value can be changed by using the [BrowserContext.setDefaultTimeout(timeout)](./api/class-browsercontext.mdx#browser-context-set-default-timeout).<a href="#web-socket-wait-for-frame-received-option-timeout" class="list-anchor">#</a>
- `callback`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-option-callback"/> &#60;[Runnable]&#62; Callback that performs the action triggering the event.<a href="#web-socket-wait-for-frame-received-option-callback" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-return"/> &#60;[WebSocketFrame]&#62;<a href="#web-socket-wait-for-frame-received-return" class="list-anchor">#</a>

Performs action and waits for a frame to be sent. If predicate is provided, it passes [WebSocketFrame] value into the `predicate` function and waits for `predicate(webSocketFrame)` to return a truthy value. Will throw an error if the WebSocket or Page is closed before the frame is received.
