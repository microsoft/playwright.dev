The [PageAssertions] class provides assertion methods that can be used to make assertions about the [Page] state in the tests. A new instance of [LocatorAssertions] is created by calling [PlaywrightAssertions.assertThat(page)](./api/class-playwrightassertions.mdx#playwright-assertions-assert-that-page):

```java
...
import static com.microsoft.playwright.assertions.PlaywrightAssertions.assertThat;

public class TestPage {
  ...
  @Test
  void navigatesToLoginPage() {
    ...
    page.click("#login");
    assertThat(page).hasURL(Pattern.compile(".*/login"));
  }
}
```

- [PageAssertions.hasURL(urlOrRegExp[, options])](./api/class-pageassertions.mdx#page-assertions-has-url)
- [PageAssertions.not()](./api/class-pageassertions.mdx#page-assertions-not)

## PageAssertions.hasTitle(titleOrRegExp[, options]) {#page-assertions-has-title}
- `titleOrRegExp`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-has-title-option-title-or-reg-exp"/> &#60;[String]|[Pattern]&#62; Expected title or RegExp.<a href="#page-assertions-has-title-option-title-or-reg-exp" class="list-anchor">#</a>
- `options` &#60;`PageAssertions.HasTitleOptions`&#62;
  - `setTimeout`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-has-title-option-timeout"/> &#60;[double]&#62; Time to retry the assertion for.<a href="#page-assertions-has-title-option-timeout" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-has-title-return"/> &#60;[void]&#62;<a href="#page-assertions-has-title-return" class="list-anchor">#</a>

Ensures the page has the given title.

```java
assertThat(page).hasTitle("Playwright");
```

## PageAssertions.hasURL(urlOrRegExp[, options]) {#page-assertions-has-url}
- `urlOrRegExp`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-has-url-option-url-or-reg-exp"/> &#60;[String]|[Pattern]&#62; Expected substring or RegExp.<a href="#page-assertions-has-url-option-url-or-reg-exp" class="list-anchor">#</a>
- `options` &#60;`PageAssertions.HasURLOptions`&#62;
  - `setTimeout`<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-has-url-option-timeout"/> &#60;[double]&#62; Time to retry the assertion for.<a href="#page-assertions-has-url-option-timeout" class="list-anchor">#</a>
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-has-url-return"/> &#60;[void]&#62;<a href="#page-assertions-has-url-return" class="list-anchor">#</a>

Ensures the page is navigated to the given URL.

```java
assertThat(page).hasURL(".com");
```

## PageAssertions.not() {#page-assertions-not}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="page-assertions-not-return"/> &#60;[PageAssertions]&#62;<a href="#page-assertions-not-return" class="list-anchor">#</a>

Makes the assertion check for the opposite condition. For example, this code tests that the page URL doesn't contain `"error"`:

```java
assertThat(page).not().hasURL("error");
```
