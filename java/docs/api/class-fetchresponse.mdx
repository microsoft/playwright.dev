[FetchResponse] class represents responses received from [BrowserContext.fetch(urlOrRequest[, options])](./api/class-browsercontext.mdx#browser-context-fetch) and [Page.fetch(urlOrRequest[, options])](./api/class-page.mdx#page-fetch) methods.


- [FetchResponse.body()](./api/class-fetchresponse.mdx#fetch-response-body)
- [FetchResponse.dispose()](./api/class-fetchresponse.mdx#fetch-response-dispose)
- [FetchResponse.headers()](./api/class-fetchresponse.mdx#fetch-response-headers)
- [FetchResponse.headersArray()](./api/class-fetchresponse.mdx#fetch-response-headers-array)
- [FetchResponse.ok()](./api/class-fetchresponse.mdx#fetch-response-ok)
- [FetchResponse.status()](./api/class-fetchresponse.mdx#fetch-response-status)
- [FetchResponse.statusText()](./api/class-fetchresponse.mdx#fetch-response-status-text)
- [FetchResponse.text()](./api/class-fetchresponse.mdx#fetch-response-text)
- [FetchResponse.url()](./api/class-fetchresponse.mdx#fetch-response-url)

## FetchResponse.body() {#fetch-response-body}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-body-return"/> &#60;[byte&#91;&#93;]&#62;<a href="#fetch-response-body-return" class="list-anchor">#</a>

Returns the buffer with response body.

## FetchResponse.dispose() {#fetch-response-dispose}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-dispose-return"/> &#60;[void]&#62;<a href="#fetch-response-dispose-return" class="list-anchor">#</a>

Disposes the body of this response. If not called then the body will stay in memory until the context closes.

## FetchResponse.headers() {#fetch-response-headers}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-headers-return"/> &#60;[Map]<[String], [String]>&#62;<a href="#fetch-response-headers-return" class="list-anchor">#</a>

An object with all the response HTTP headers associated with this response.

## FetchResponse.headersArray() {#fetch-response-headers-array}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-headers-array-return"/> &#60;[List]<`HttpHeader`>&#62;<a href="#fetch-response-headers-array-return" class="list-anchor">#</a>
  - `name` &#60;[String]&#62; Name of the header.
  - `value` &#60;[String]&#62; Value of the header.

An array with all the request HTTP headers associated with this response. Header names are not lower-cased. Headers with multiple entries, such as `Set-Cookie`, appear in the array multiple times.

## FetchResponse.ok() {#fetch-response-ok}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-ok-return"/> &#60;[boolean]&#62;<a href="#fetch-response-ok-return" class="list-anchor">#</a>

Contains a boolean stating whether the response was successful (status in the range 200-299) or not.

## FetchResponse.status() {#fetch-response-status}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-status-return"/> &#60;[int]&#62;<a href="#fetch-response-status-return" class="list-anchor">#</a>

Contains the status code of the response (e.g., 200 for a success).

## FetchResponse.statusText() {#fetch-response-status-text}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-status-text-return"/> &#60;[String]&#62;<a href="#fetch-response-status-text-return" class="list-anchor">#</a>

Contains the status text of the response (e.g. usually an "OK" for a success).

## FetchResponse.text() {#fetch-response-text}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-text-return"/> &#60;[String]&#62;<a href="#fetch-response-text-return" class="list-anchor">#</a>

Returns the text representation of response body.

## FetchResponse.url() {#fetch-response-url}
- returns:<a aria-hidden="true" tabindex="-1" class="list-anchor-link" id="fetch-response-url-return"/> &#60;[String]&#62;<a href="#fetch-response-url-return" class="list-anchor">#</a>

Contains the URL of the response.
