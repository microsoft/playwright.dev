API for controlling the Playwright debugger. The debugger allows pausing script execution and inspecting the page. Obtain the debugger instance via [BrowserContext.debugger()](/api/class-browsercontext.mdx#browser-context-debugger).


---

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.next</x-search>

**Usage**

```java
Debugger.next();
```

### pause {#debugger-pause}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.pause</x-search>

Configures the debugger to pause before the next action is executed.

Throws if the debugger is already paused. Use [Debugger.next()](/api/class-debugger.mdx#debugger-next) or [Debugger.runTo()](/api/class-debugger.mdx#debugger-run-to) to step while paused.

Note that [Page.pause()](/api/class-page.mdx#page-pause) is equivalent to a "debugger" statement — it pauses execution at the call site immediately. On the contrary, [Debugger.pause()](/api/class-debugger.mdx#debugger-pause) is equivalent to "pause on next statement" — it configures the debugger to pause before the next action is executed.

**Usage**

```java
Debugger.pause();
```

**Returns**
- [void]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-pause-return"/><a href="#debugger-pause-return" class="list-anchor">#</a>

### runTo {#debugger-run-to}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.59</font><x-search>debugger.runTo</x-search>

Resumes script execution and pauses when an action originates from the given source location. Throws if the debugger is not paused.

**Usage**

```java
Debugger.runTo(location);
```

**Arguments**
- `location` Location<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-run-to-option-location"/><a href="#debugger-run-to-option-location" class="list-anchor">#</a>
  - `setFile` [String]
    
    
  - `setLine` [int] *(optional)*
    
    
  - `setColumn` [int] *(optional)*
    
    
  The source location to pause at.

**Returns**
- [void]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="debugger-run-to-return"/><a href="#debugger-run-to-return" class="list-anchor">#</a>
