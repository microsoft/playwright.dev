Playwright module provides a method to launch a browser instance. The following is a typical example of using Playwright to drive automation:

```java
import com.microsoft.playwright.*;

public class Example {
  public static void main(String[] args) {
    try (Playwright playwright = Playwright.create()) {
      BrowserType chromium = playwright.chromium();
      Browser browser = chromium.launch();
      Page page = browser.newPage();
      page.navigate("http://example.com");
      // other actions...
      browser.close();
    }
  }
}
```

### close {#playwright-close}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>playwright.close</x-search>

**Usage**

```java
Playwright.close();
```

---

### create {#playwright-create}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>playwright.create</x-search>

Launches new Playwright driver process and connects to it. [Playwright.close()](/api/class-playwright.mdx#playwright-close) should be called when the instance is no longer needed.

```java
Playwright playwright = Playwright.create();
Browser browser = playwright.webkit().launch();
Page page = browser.newPage();
page.navigate("https://www.w3.org/");
playwright.close();
```

**Usage**

```java
Playwright.create();
Playwright.create(options);
```

**Arguments**
- `options` `Playwright.CreateOptions` *(optional)*
  - `setEnv` [Map]&lt;[String], [String]&gt; *(optional)* <font size="2">Added in: v1.13</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="playwright-create-option-env"/><a href="#playwright-create-option-env" class="list-anchor">#</a>
    
    Additional environment variables that will be passed to the driver process. By default driver process inherits environment variables of the Playwright process.

**Returns**
- [Playwright]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="playwright-create-return"/><a href="#playwright-create-return" class="list-anchor">#</a>
