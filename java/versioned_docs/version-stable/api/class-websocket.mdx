### waitForFrameReceived {#web-socket-wait-for-frame-received}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>webSocket.waitForFrameReceived</x-search>

Performs action and waits for a frame to be sent. If predicate is provided, it passes [WebSocketFrame] value into the `predicate` function and waits for `predicate(webSocketFrame)` to return a truthy value. Will throw an error if the WebSocket or Page is closed before the frame is received.

**Usage**

```java
WebSocket.waitForFrameReceived(callback);
WebSocket.waitForFrameReceived(callback, options);
```

**Arguments**
- `options` `WebSocket.WaitForFrameReceivedOptions` *(optional)*
  - `setPredicate` [Predicate]&lt;[WebSocketFrame]&gt; *(optional)* <font size="2">Added in: v1.9</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-option-predicate"/><a href="#web-socket-wait-for-frame-received-option-predicate" class="list-anchor">#</a>
    
    Receives the [WebSocketFrame] object and resolves to truthy value when the waiting should resolve.
  - `setTimeout` [double] *(optional)* <font size="2">Added in: v1.9</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-option-timeout"/><a href="#web-socket-wait-for-frame-received-option-timeout" class="list-anchor">#</a>
    
    Maximum time to wait for in milliseconds. Defaults to `30000` (30 seconds). Pass `0` to disable timeout. The default value can be changed by using the [BrowserContext.setDefaultTimeout()](/api/class-browsercontext.mdx#browser-context-set-default-timeout).
- `callback` [Runnable] <font size="2">Added in: v1.9</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-option-callback"/><a href="#web-socket-wait-for-frame-received-option-callback" class="list-anchor">#</a>
  
  Callback that performs the action triggering the event.

**Returns**
- [WebSocketFrame]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="web-socket-wait-for-frame-received-return"/><a href="#web-socket-wait-for-frame-received-return" class="list-anchor">#</a>
